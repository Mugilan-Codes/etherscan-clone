# etherscan-clone

## A Etherscan Clone using Vue CLI and Web3.js

### Scripts (package.json)

- Install Packages

```sh
npm install
```

- Compiles and hot-reloads for development

```sh
npm run serve
```

- Compiles and minifies for production

```sh
npm run build
```

- Lints and fixes files

```sh
npm run lint
```
